# 🎪 Gestion Interactifs Asso

Application de gestion complète pour associations, développée avec Python et CustomTkinter.

## Fonctionnalités

- 🏠 Tableau de bord avec alertes
- 🎪 Gestion des événements (billetterie, dépenses, bénévoles, tombola)
- 💰 Trésorerie (comptes, opérations, remises, subventions)
- 📦 Stock & Buvette (articles, entrées, inventaires)
- 👥 Adhérents & Cotisations
- 📄 Exports & Bilan AG en PDF
- ⚙️ Paramètres & Sauvegardes automatiques

## Prérequis

- Python 3.10 ou supérieur
- tkinter (inclus sur Windows ; sur Linux : `sudo apt install python3-tk`)

## Installation

1. Télécharger et décompresser le projet
2. Installer les dépendances :
   ```
   pip install -r requirements.txt
   ```
3. Lancer l'application :
   ```
   python run_app.py
   ```

## Premier démarrage

Au premier lancement, l'application vous demande de créer ou sélectionner
une base de données. Consultez le Manuel d'utilisation (`docs/MANUEL_UTILISATION.md`)
pour une prise en main complète.
